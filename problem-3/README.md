# Internet Companion

This is an extension for all people who wish to have a companion with them while browsing the web.
With **Internet Companion** you won't ever be alone again as you're scrolling. The companion will even help you find cool videos to watch and may replace some images to remind you they are there with you!

## Features:
- Replaces all link (**a** tags) hrefs with a link to a *spooky* video
- Replaces all images with a random image from a list of *spooky* images
- Replaces all **h1** tags with creepy lines from our list of creepy sentences
- Replaces all **p** tags in the same manner
- Replaces all **span** tags in the same manner
- All **h1** and **div** tags have their background colours set to red to really give a nice atmosphere
